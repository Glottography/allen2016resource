# Glottography dataset derived from Allen et al. 2016 "Resource scarcity drives lethal aggression among prehistoric hunter-gatherers in central California"

[![CLDF validation](https://github.com/Glottography/allen2016resource/workflows/CLDF-validation/badge.svg)](https://github.com/Glottography/allen2016resource/actions?query=workflow%3ACLDF-validation)

## How to cite

If you use these data please cite
- the original source
  > M.W. Allen, R.L. Bettinger, B.F. Codding, T.L. Jones, A.W. Schwitalla, Resource scarcity drives lethal aggression among prehistoric hunter-gatherers in central California, Proc. Natl. Acad. Sci. U.S.A. 113 (43) 12120-12125, https://doi.org/10.1073/pnas.1607996113 (2016).
- the derived dataset using the DOI of the [particular released version](../../releases/) you were using

## Description


This dataset provides non-overlapping, contact-era ethnolinguistic areas in central California as described in Fig. 1 in Allen et al. 2016.
